Number Formatting

Formatting the number in currency comma format.

Usage:

  • include "numberFormatting" js & css in header
  • define input field inside the div
  • call doRegisterPcrmNumberFormattingEvents() function after DOM rendered
  • configure configNumberFormatting() for the respective input field, wherever need. 

    <head>
        <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js" type="text/javascript"></script>
        <script src="js/numberFormatting.js" type="text/javascript"></script>
        <link rel="stylesheet" href="css/numberFormatting.css">
    </head>
    <body>
        <div>
            <input type="text" id="id_inputField" />
        </div>
    </body>
    <script>
        $(function() {
            doRegisterPcrmNumberFormattingEvents();
            $("#id_inputField").configNumberFormatting();
        })
    </script>

Options:

The options that you can set are:

  • nfInputSign: Currency format. It will appear as number prefix. (Default value: '')
  • nfIsAllowDecimal: Allow to enter decimal value. (Default value: false)
  • nfDecimalMinLength: Define decimal precision minimum length. (Default value: 0)
  • nfDecimalMaxLength: Define decimal precision maximum length. (Default value: 20)
  • nfIsAllowNegative: Allow to enter negative value. (Default value: false)
  • nfMinNumber: Define the minimum value. (Default value: 0)
  • nfMaxNumber: Define the maximum value. (Default value: '')
  • nfStepValue: Define the step value while clicking up/down arrow (keyboard key / button). (Default value: '', if nfInputArrowBtn is true, then default value: '0.1')
  • nfInputArrowBtn: Show the up & down arrow button. (Default value: false)

⚠ Important Note: This number formatting plugin is in-development. Currently this plugin is developed for US-Dollar currency only. Will update the same plugin for accepting all the currency format and with more other options.

Working Examples:

Empty Initialize 

$("#id_inputField_1").configNumberFormatting();

It just doing formatting number in Indian current format.

nfInputSign 

$("#id_inputField_2").configNumberFormatting({ nfInputSign: "$" });

Current format. It will appear as number prefix. (Default value: '')

nfIsAllowDecimal 

$("#id_inputField_3").configNumberFormatting({ nfIsAllowDecimal: "true" });

Allow to enter decimal value. (Default value: false)

nfDecimalMinLength 

$("#id_inputField_4").configNumberFormatting({ nfDecimalMinLength: "1" });

Define decimal precision minimum length. (Default value: 0)

nfDecimalMaxLength 

$("#id_inputField_5").configNumberFormatting({ nfDecimalMaxLength: "2" });

Define decimal precision maximum length. (Default value: 20)

nfIsAllowNegative 

$("#id_inputField_6").configNumberFormatting({ nfIsAllowNegative: "true" });

Allow to enter negative value. (Default value: false)

nfMinNumber 

$("#id_inputField_7").configNumberFormatting({ nfMinNumber: "10" });

Define the minimum value. (Default value: 0)

nfMaxNumber 

$("#id_inputField_8").configNumberFormatting({ nfMaxNumber: "1000" });

Define the maximum value. (Default value: '')

nfStepValue 

$("#id_inputField_9").configNumberFormatting({ nfStepValue: "0.5" });

Define the step value while clicking up/down arrow (keyboard key / button). (Default value: '', if nfInputArrowBtn is true, then default value: '0.1')

nfInputArrowBtn 

$("#id_inputField_10").configNumberFormatting({ nfInputArrowBtn: "true" });

Show the up & down arrow button. (Default value: false)
